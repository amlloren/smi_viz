# Project Overview:


## Data Cleaning:

### Renamed datasets into a uniformed nomenclature

| Platform | Shortcut | Uniformed Nomenclature |
|----------|----------|------------------------|
| Instagram | IG | smi_YYYY_MM_IG.csv |
| TikTok | TT | smi_YYYY_MM_TT.csv |
| YouTube | YT | smi_YYYY_MM_YT.csv |
